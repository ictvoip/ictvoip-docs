Client Portal Addon
==================

Overview
--------

The Client Portal addon provides customers with a comprehensive web interface to manage their VoIP services, view billing information, and access call details.

Features
--------

* **Call History** - View detailed call logs and CDR data
* **Billing Information** - Access invoices and payment history
* **Service Management** - Manage extensions and services
* **Usage Reports** - Detailed usage analytics
* **Payment Processing** - Online payment capabilities
* **Support Tickets** - Integrated support system
